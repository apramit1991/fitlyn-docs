# Fitlyn Public Documentation

Public, customer-safe documentation for Fitlyn.

The complete internal engineering documentation remains in the private
application repository. Do not copy architecture, database, security,
credentials, customer data, internal URLs, incident reports, or deployment
procedures into this repository.

## Local preview

```bash
python -m pip install -r requirements.txt
mkdocs serve
```

## Publishing

Changes merged into `main` are built and published through GitHub Pages.
Follow [CONTRIBUTING.md](CONTRIBUTING.md) for the update and safety-review
process.
