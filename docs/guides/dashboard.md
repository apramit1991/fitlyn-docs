# Dashboard

The dashboard provides a quick view of gym activity.

Depending on your role and enabled modules, it may show:

- active and recently added members;
- current-month collections and pending dues;
- upcoming membership expiries;
- recent payments;
- birthdays and other follow-up reminders;
- shortcuts to common actions.

## Recommended routine

1. Check expiring memberships at the start of the day.
2. Review pending dues before contacting members.
3. Reconcile displayed collections with payment entries.
4. Follow up on unexpected changes before adding corrective entries.

Dashboard figures summarize recorded data. If a figure appears wrong, first
check payment dates, membership dates, status, and the selected gym.
