# Attendance

Fitlyn supports manual attendance and supported biometric integrations.

## Manual check-in

1. Open **Attendance**.
2. Search for the member.
3. Confirm the correct profile.
4. Record the check-in.

Only active members should be checked in.

## Biometric attendance

Biometric setup may require an on-site device, member-to-device mapping, and
Fitlyn support or administrator access.

If a punch is missing:

1. Confirm the member is active.
2. Confirm the device identifier is mapped to the correct member.
3. Check device connectivity.
4. Review pending or failed synchronization information.
5. Record attendance manually only after confirming it will not create a
   duplicate.
