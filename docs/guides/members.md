# Members

The Members area stores each member's profile and activity context.

## Add a member

1. Open **Members**.
2. Select **Add member**.
3. Enter the required identity and contact information.
4. Capture communication consent accurately.
5. Save the profile.

Phone numbers may be used for duplicate detection. Verify the number before
creating another profile.

## Manage a member

From a member profile you can review available information such as:

- profile and contact details;
- memberships and renewal dates;
- invoices, payments, and dues;
- attendance;
- personal-training information, when enabled.

## Deactivate instead of deleting

When a member leaves, mark the profile inactive. This preserves financial and
attendance history.

## Bulk import

Use the importer only with a clean test file first. Validate names, phone
numbers, dates, plans, amounts, and consent fields. Review the import result and
correct rejected rows before retrying.
